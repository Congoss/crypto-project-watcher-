# Crypto Project Watcher

Use this skill to produce neutral, structured crypto project reports from a project name.

## Output

Project: {name}

Overview:
- short description

Activity Analysis:
- development activity level reasoning

Signals:
Positive:
- generic signals

Risks:
- generic risks

Conclusion:
- neutral summary

## Rules

- Do not invent financial data or news.
- Keep analysis conceptual and generic.
- If evidence is unavailable, state that clearly.
